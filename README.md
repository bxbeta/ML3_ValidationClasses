## 📊 ML3: Методы валидации моделей. Реализация классов для разбиения признаков и валидации моделей

### 📌 О чем проект?
Проект посвящён реализации методов валидации моделей и отбора признаков с нуля. 
Основной фокус — построить собственные инструменты для разбиения данных и проверки качества моделей.

---

### 🚀 Что сделано
- Реализованы классы для работы с данными:
  - `DataSplit` — разбиение на обучающую и тестовую выборки
  - `Validation` — запуск различных стратегий валидации
- Написан простой алгоритм отбора признаков на основе корреляции
- Проведена оценка важности признаков (перестановки, SHAP)
- Обучены модели и проведено сравнение качества

---

### ⚙️ Реализация
- Вся логика валидации и разбиения данных реализована с нуля без использования встроенных реализаций из sklearn
- Используются базовые библиотеки (`numpy`, `pandas`)
- Дополнительно применяются инструменты визуализации и анализа (`matplotlib`, `seaborn`, `shap`)
- Код структурирован через классы и функции для повторного использования
