## 📊 ML3: Методы валидации моделей. Реализация классов GridSearch

### 📌 О чем проект?
Проект посвящён реализации методов валидации моделей и отбора признаков с нуля. 
Основной фокус — построить собственные инструменты для разбиения данных, проверки моделей и подбора параметров.

---

### 🚀 Что сделано
- Реализованы классы для работы с данными:
  - `DataSplit` — разбиение на обучающую и тестовую выборки
  - `Validation` — запуск различных стратегий валидации (включая Leave-One-Out)
- Реализованы методы подбора параметров:
  - аналог Grid Search
  - Random Search CV
- Написан простой алгоритм отбора признаков на основе корреляции
- Проведена оценка важности признаков (перестановки, SHAP)
- Обучены модели и проведено сравнение качества

---

### ⚙️ Реализация
- Вся логика валидации и разбиения данных реализована вручную
- Используются базовые библиотеки (`numpy`, `pandas`)
- Дополнительно применяются инструменты визуализации и анализа (`matplotlib`, `seaborn`, `shap`)
- Код структурирован через классы и функции для повторного использования

---

### 📈 Результат
Проект демонстрирует:
- понимание принципов валидации моделей
- умение реализовывать базовые инструменты анализа данных
- работу с отбором признаков и оценкой их влияния на модель
